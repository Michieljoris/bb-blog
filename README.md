bb-blog
--------

[Description]

Install:

    npm install bb-blog
	
Add dependency to your project with

    "bb-blog": "git@github.com/Michieljoris/bb-blog"
	
or

	"bb-blog": "*"

Require in a module:

    var bb-blog = require('bb-blog');

Use:

See [documentation](https://rawgithub.com/Michieljoris/bb-blog/master/docs/bb-blog.html).
